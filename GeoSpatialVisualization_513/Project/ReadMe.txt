Below are the steps to run our project:

Pre-Requisites:
You will need below libraries to run the code.
1. Math
2. PCL
3. Numpy

Once your platform is equipped with above libraries you can run main.py file.

Input:
final_project_point_cloud.fuse

Outputs:
It will create four obj files as an output for this project.
1.  Original Point Cloud.obj
2.  Intermediate_File1.obj
3.  Intermediate_File2.obj
4.  Final_File.obj

To view this obj files in a 3-D point, Please open these files in meshlab software.
